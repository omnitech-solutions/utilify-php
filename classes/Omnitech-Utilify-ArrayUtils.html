Filters the attributes based on the given filters.

public static filterAttributesByConditions(array<string, mixed> $attributes, array<string, bool|string|callable> $filters) : array<string, mixed>

Each filter must have a key and a value of either a string, a regular expression, or a callable that returns a boolean.

Parameters
$attributes : array<string, mixed>

The array of attributes to be filtered (string keys, mixed values).

$filters : array<string, bool|string|callable>

An array of filters to apply. Filters can be a boolean, string, regex pattern, or a callable.

Tags
example

['field_1' => true] // Always includes 'field_1' ['field_2' => /^[0-9]*$/] // Includes 'field_2' if it contains only numbers ['field_3' => function ($value) { // Includes 'field_3' if the callback evaluates to true return in_array($value, ['allowed_value1', 'allowed_value2'], true); }]

Return values
array<string, mixed> —

The filtered attributes that match the provided filters.

This method searches the keys of the provided array and retains only those that match the specified regular expression. The values associated with those keys are included in the returned array.

public static isBlank(mixed $value) : bool

This function determines whether the provided value is either a non-numeric blank value (empty or null) or a numeric value that is equal to 0.

  • If the value is non-numeric, the function checks if it is blank (empty).
  • If the value is numeric, it returns true if the value is exactly zero.
Parameters
$value : mixed

The value to check, which can be of any type (string, number, etc.).

Return values
bool —

Returns true if the value is filled or not equal to 0, otherwise false.

public static isFilled(mixed $value) : bool

This function determines whether the provided value is either a non-numeric filled value (not empty or null) or a numeric value that is not equal to 0.
